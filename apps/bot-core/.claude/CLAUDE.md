- Discord bot app to allow interaction through Discord - used primarily for receiving events and running commands. Bot runs on multiple shards & clusters to handle large scale.

# Project tech stack
- discord.js - Framework for Discord API - https://discord.js.org/docs/packages/discord.js/main
- Sapphire - Framework for building Discord bots using discord.js - https://sapphirejs.dev/docs/General/Welcome
- discord-hybrid-sharding - Library for sharding Discord bots - https://github.com/meister03/discord-hybrid-sharding/blob/ts-rewrite/README.md
- Uses apps/bot-api as internal API to send requests to Discord API.
